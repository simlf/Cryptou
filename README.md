# Cryptou

## Quick Start

Versions

```
yarn -v
 -> 1.22.19
node -v
 -> v16.17.0
```

Developpement

```
# install (new) packages
yarn

# launch server front
yarn dev
 -> Local: http://127.0.0.1:5173/

# launch server back
yarn start
 -> Local: http://localhost:3000/
```
